# React Firebase Authentication Quiz

This is a simple React and Firebase authentication quiz covering key concepts about managing authentication state, using Firebase methods, and React hooks.

---

### **Question 01**

**What is the main purpose of creating a User Context in React Authentication?**

* **Answer:** To share the user authentication state across the app

---

### **Question 02**

**Which hook is used to access data from Context in React?**

* **Answer:** `useContext`

### **Question 04**

**What is a Private Route in React?**

* **Answer:** A route that requires a logged-in user to access

---

### **Question 05**

**Which Firebase method is used to create a new user with email and password?**

* **Answer:** `createUserWithEmailAndPassword()`

---

### **Question 06**

**Which Firebase method is used to sign in a user with email and password?**

* **Answer:** `signInWithEmailAndPassword()`

---

### **Question 07**

**What should you do after a successful login using Firebase in React?**

* **Answer:** Navigate the user to a protected route

---

### **Question 08**

**What is the purpose of handling a loading state in authentication?**

* **Answer:** To show a spinner or loader while checking the user status

---

### **Question 09**

**What does the `signOut()` function do in Firebase?**

* **Answer:** Logs the user out of the current session

---

### **Question 10**

**What is the benefit of using Context API for authentication in React?**

* **Answer:** Avoids prop drilling and provides global user access
